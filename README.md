# LinkedIn Post Generator (AI Agent)

An AI-powered LinkedIn post generator built using LangChain and Gemma3:4B via Ollama.

## Features
- Topic-based post generation
- Multiple tones (Professional, Casual, Storytelling, etc.)
- Multi-language support (English, Bengali, Spanish)
- Local LLM inference 

## Tech Stack
- Python
- LangChain (LCEL)
- Ollama
- Gemma3:4B
- Google Colab

## How it works
User input → Prompt template → LangChain pipeline → Local LLM → LinkedIn-ready post
